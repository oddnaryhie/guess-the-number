# Number Guessing Game

## Features

- Computer generates a random number  
- Player tries to guess the number  
- Gives hints like "Too high" or "Too low"  
- Tracks number of attempts  
- Play Again option  

## How to Run

- Open project folder
- Run main.py using Python

## Concepts Used

- Loops  
- If-else conditions  
- Random module  
- User input
